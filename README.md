# TFM---Big-Data-Aplicado-al-Deporte
TFM del Máster de Big Data Aplicado al Deporte por de Sports Datacampus por la UCAM

El proyecto consiste en el desarrollo de una aplicación de escritorio en Python 
que automatiza la obtención, visualización y análisis de estadísticas de equipos 
y jugadores de la Federació Catalana de Basquetbol (FCBQ). La herramienta 
centraliza datos dispersos, facilita su análisis y presenta visualizaciones avanzadas 
como gráficos de barras y radar, permitiendo un análisis más profundo y eficiente 
del rendimiento deportivo.
